# Projeto IoT Big Data FECAF

Pipeline de dados IoT com Python, PostgreSQL e Streamlit.

## Estrutura
- src/main.py
- src/dashboard.py
- src/views.sql
- data/iot_temperature.csv
- .env.example
- requirements.txt

## Execucao local
1. Criar e ativar venv
2. Instalar requirements
3. Subir PostgreSQL
4. Rodar pipeline
5. Rodar streamlit
